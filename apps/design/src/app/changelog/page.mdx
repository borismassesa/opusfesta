## v2026.04 · initial release

**18 April 2026**

First cut of the OpusFesta design system, extracted from the live marketing site at `apps/opus_website`.

- **Added** — full foundations (colour, type, spacing, radius, elevation, motion, iconography).
- **Added** — first components: Button, Pill &amp; Badge, Card, Input.
- **Added** — first patterns: Hero, Dark section, Full-bleed CTA, FAQ, Testimonial.
- **Added** — voice &amp; tone guidelines with Tanzania-first locale cues.
- **Established** — lavender `#C9A0DC` as the single brand accent, ink `#1A1A1A` for everything primary, full-rounded buttons, soft shadows, UPPERCASE font-black display type.
- **Closed** — category palette at 26 colours.

**Canonical source:** `apps/opus_website` landing page.

## Writing a changelog entry

1. Edit `apps/design/src/app/changelog/page.mdx`.
2. Add a new `## vYYYY.MM · summary` section at the top.
3. Group changes under **Added**, **Changed**, **Removed**, **Fixed**.
4. Call out breaking changes in bold.
5. Link the PR if one exists.
