## Accessibility

- Every input needs a visible label or a `kicker` styled eyebrow above.
- Focus ring: `outline outline-4 outline-[var(--accent)]/30`.
- Error state: `border-red-400` + red-500 body text below.
- Never hide labels behind placeholder text alone.

<DosAndDonts>
  <Do>Use the kicker pattern for field labels — 10–11px, uppercase, tracking-widest.</Do>
  <Dont>Use default browser inputs — they break the visual language.</Dont>
  <Do>Group related fields inside a single rounded-2xl container with divider rules.</Do>
  <Dont>Use placeholder text as the only label — fails accessibility.</Dont>
</DosAndDonts>
