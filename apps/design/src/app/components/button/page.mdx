# Button

Every button is a pill. Fully rounded, bold label, no icon unless the action needs one. Three levels: **accent** (primary), **ink** (secondary high-emphasis), **outline** (tertiary). Use an underlined text link for low-emphasis navigation.

## Accessibility

- Minimum tap target 44×44px on mobile (`md` size and up satisfies this).
- Never rely on colour alone — pair with label copy.
- Focus ring: `outline outline-4 outline-[var(--accent)]/30` — visible on all backgrounds.
- Disabled state must never look like default state.

<DosAndDonts>
  <Do>Label with 2–4 words, imperative verb first: "Start planning", "Join as a vendor".</Do>
  <Dont>Use "Click here" or "Submit". Always describe the action.</Dont>
  <Do>Place accent and ink buttons side-by-side with accent leading.</Do>
  <Dont>Put two accent buttons in the same view — lavender is for one moment per view.</Dont>
</DosAndDonts>
