# Open Graph Image for Admin Portal

## Required Image

Create an Open Graph image at: `apps/admin/public/opengraph-admin.png`

## Specifications

- **Dimensions**: 1200x630 pixels (recommended OG image size)
- **Format**: PNG or JPG
- **Content**: Should include:
  - TheFesta logo/branding
  - "Admin Portal" text
  - Professional, clean design
  - Matches TheFesta brand colors

## Quick Creation

You can:
1. Copy `apps/website/public/opengraph.png` and modify it
2. Use a design tool (Figma, Canva, etc.) to create a new one
3. Use an online OG image generator

The image will be used when sharing links to the admin portal on social media, messaging apps, etc.
